# 米塔彩铅主题设计（v2）

## 目标

在 AgentBattery Flutter 现有三主题基础上新增第四主题“米塔彩铅”。该主题使用用户提供的米塔彩铅参考图衍生出的横向背景，保持角色辨识度与传统彩铅纸纹；数据可读性优先。

## 视觉规则

- 主题名：`米塔彩铅`；内部枚举：`mita`。
- 背景资源：`assets/images/mita_colored_pencil_dashboard_background.png`。
- 横向窗口策略：目标 `1320 × 760`，最小 `1080 × 680`，与 MIKU 舞台共享横向窗口规格。
- 构图：右侧约 45% 为米塔人物和暖紫小屋背景；左侧约 55% 为数据区。
- 遮罩：左侧深靛紫→半透明紫蓝渐隐，防止彩铅纹理干扰数据。
- 卡片：暖白、淡紫、半透明纸感，带极轻微紫色边框与阴影。
- 色彩：深靛紫、蓝紫、绯红、暖粉、少量金色星点；避免与可爱风的桃粉视觉混淆。
- 文案：克制采用“彩铅小屋”“小伙伴状态簿”“能量小金库”等，不将核心财务数据名称改得难理解。

## 交互与响应式

- 主题菜单新增“米塔彩铅”。
- 主题持久化；旧主题解析规则不变。
- 宽度 < 920px 时使用紧凑/纵向流，人物背景右裁切、数据优先。
- 玻璃、可爱风、MIKU 既有窗口策略与视觉效果不得回归。

## 验证

- 增加 AppTheme 持久化和横向窗口策略测试。
- 新增米塔宽屏与窄屏 Widget 测试，不可 overflow。
- 运行 `flutter test`、`flutter analyze`、`flutter build windows --debug`。
- 实际 Windows 截图验收：右侧人物完整，左侧数据区不遮挡，彩铅纸纹与暖白紫卡片可辨。
